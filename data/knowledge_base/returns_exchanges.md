# Returns and Exchanges

Items can be returned or exchanged within 30 days of delivery if unused and in
original packaging. The customer must initiate a return request through the
order portal to receive a prepaid return label.

Exchanges for a different size or color are processed as soon as the returned
item is scanned by the carrier - the replacement ships immediately rather than
waiting for the return to arrive at the warehouse.

Final sale and clearance items are not eligible for return or exchange unless
defective. Defective items can be returned at any time within the 1-year
warranty period for a full refund or replacement.

Common triggers: "return this item", "exchange for a different size", "wrong
item received", "defective product", "warranty claim".
